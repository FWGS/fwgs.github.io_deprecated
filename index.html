Our projects

Xash3D SDL

Xash3D engine ported to SDL. Works even on your phone!

CS1.6 Client

Counter-Strike 1.6 client.dll rewrite project.

SDLash HLSDK and SDLash SoHL

HLSDKs ported to Android and ARM.

NanoGL

Super fast OpenGL-to-OpenGLES translator. Originally written by Olli Hinkka.

XashDS Android

Android dedicated server with x86 dynamic translator and DLL support.

MainUI C++

GUI library for Xash3D, written in C++. Includes window system, powerful font rendering and dynamic menus. Supports both original Xash3D and Xash3D FWGS.

Source SDK 2013 Tegra

Source SDK 2013 ported to Android for NVIDIA Shield port and some utilities for it, like touch controls, fixes for game console and ultimate solution for wrapping libraries with Valve interface.
